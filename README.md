# biblio_BFS

Projet Python autour d’un algorithme de recherche en largeur (**BFS**).

## À propos

Ce dépôt m’a servi à explorer la logique des parcours de graphe et de recherche de chemin à travers de petits essais en Python.

L’idée n’était pas seulement d’obtenir un résultat, mais surtout de comprendre comment représenter un problème, parcourir les états possibles et structurer proprement la recherche.

## Ce que j’y ai appris

- modéliser un espace de recherche
- appliquer un parcours en largeur
- manipuler des structures utiles comme les files
- raisonner sur les chemins, les nœuds et les états
- organiser un petit projet Python de manière plus claire

## Contenu du dépôt

Le dépôt contient plusieurs fichiers liés à mes expérimentations autour du BFS et de la recherche de chemin.
